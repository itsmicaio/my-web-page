Estou no terceiro dia do [Hot n' Code](https://campaign.hotmart.com/hotncode-conference) um evento incrível que está me dando váris insights sobre o mundo da tecnologia. Hoje eu vou falar sobre o que eu aprendi no terceiro dia do evento, que foi sobre Data And Artificial Intelligence.

## AWS Bedrock
A solução do da Amazon para IA Generativas, o AWS Bedrock é um conjunto de ferramentas que permite a criação de modelos de IA Generativa, que são modelos que conseguem gerar novos dados a partir de um conjunto de dados de treinamento. Esses modelos são capazes de gerar novas imagens, textos, sons e até mesmo vídeos.

## Amazon SageMaker
O Amazon SageMaker é um serviço totalmente gerenciado que permite que cientistas de dados e desenvolvedores criem, treinem e implantem de maneira rápida e fácil modelos de machine learning (ML) em qualquer escala. O SageMaker remove todas as barreiras que podem atrasar o desenvolvimento de modelos de ML, como provisionamento de instâncias, gerenciamento de escalabilidade e monitoramento. O SageMaker inclui módulos integrados que cobrem as etapas de fluxo de trabalho de ML, para que os desenvolvedores possam criar modelos de ML sem precisar escrever nenhum código. O SageMaker também pode ser personalizado com algoritmos de ML da AWS Marketplace e estruturas de ML populares, como TensorFlow e Apache MXNet.

## AI Explorer
O site [AI Use Case Explorer](https://aiexplorer.aws.amazon.com/) é um portal onde a gente pode explorar os modelos de IA que a Amazon disponibiliza. É possível testar os modelos de IA e ver como eles funcionam.

## Change Data Capture
Esse é um padrão de arquitetura que permite que a gente capture as mudanças que acontecem em um banco de dados, através dos `changelogs`. A ideia é que a gente consiga capturar essas mudanças e enviar para um outro banco de dados, que pode ser um banco de dados de analytics, um data lake ou até mesmo um data warehouse.
